## All Global Format:
### **Portion #1** *(Language Namespace to Language ID)*:
> All spaces in current format case is represented as **"\t"** *(Chr #9)*
```
<len6hex LANGID> <str NAMESPACE>
```

### **Portion #2** *(Language Localization Content)*:
> All spaces in current format case is represented as **"\t"** *(Chr #9)*
```
<len6hex LANGID> <str CONTENT>
```
